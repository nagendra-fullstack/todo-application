# ✅ ToDo Application

A simple and efficient ToDo Application built using **HTML, CSS, and JavaScript**, with **LocalStorage** support for data persistence.  
Users can add tasks, delete tasks, and mark tasks as completed.  
This project demonstrates strong understanding of DOM manipulation, event handling, and browser storage.

---

## ✨ Features

- ➕ **Add new tasks**
- ✔️ **Mark tasks as completed**
- ❌ **Delete tasks**
- 💾 **Persistent storage using LocalStorage**
- 🔄 **Automatic UI update**
- 🎨 **Clean and simple user interface**

---

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**  
- **LocalStorage**

---

## 📦 How to Run the Project

1. Clone the repository or download the ZIP file  
```
git clone https://github.com/nagendra-programmer/todo-application.git

```

2. Open the project folder  
3. Run the todo_html_file.html file in any browser 
4. Start adding your tasks!
